# FitCree カラーシステム定義書 (Tailwind CSS)

## 1. ブランド・役割別カラー

FitCreeでは、ユーザーのモード（Creator / Client）によってメインカラーを使い分けます。

| 役割 | 色名 (Tailwind) | Hex | 用途 |
|------|----------------|-----|------|
| Creator | orange-creator | #F0842C | クリエイターモード時のプライマリーカラー |
| Creator (Light) | orange-300 | #FDBA74 | 背景のアクセントなど |
| Creator (Dark) | orange-600 | #EA580C | CTAボタン、強調用 |
| Client | blue-500 | #3B82F6 | クライアントモード時のプライマリーカラー |
| Client (Light) | blue-300 | #93C5FD | 境界線、薄い背景など |
| Client (Dark) | blue-700 | #432dd7 | 強調用 |

## 2. テキストカラー

可読性を重視し、階層ごとに定義します。

| 用途 | 色名 (Tailwind) | Hex |
|------|----------------|-----|
| メイン本文・見出し | neutral-800 | #262626 |
| 小見出し・補足 | neutral-600 | #525252 |
| ラベル・キャプション | neutral-500 | #737373 |
| プレースホルダー | neutral-400 | #A3A3A3 |
| 反転テキスト | white | #FFFFFF |

## 3. 背景・UI要素

投稿ページなどの背景は、フォームの打ち込みやすさや可読性を重視し、ややグレーを帯びた色を使用します。

| 用途 | 色名 (Tailwind) | Hex |
|------|----------------|-----|
| ページ基本背景 | slate-100 | (投稿ページ等で使用) |
| コンテンツ背景 | white | #FFFFFF |
| 非活性エリア | neutral-100 | #F5F5F5 |
| セクション区切り | neutral-300 | #D4D4D4 |
| 入力フィールドボーダー | neutral-300 | #D4D4D4 |
| ホバー背景（軽い） | neutral-50 | #FAFAFA |
| 選択状態背景（Client） | blue-50 | #EFF6FF |
| 選択状態ボーダー（Client） | blue-500 | #3B82F6 |
| フォーカスリング（Client） | blue-500 | #3B82F6 |
| ステップタイトル背景 | slate-500 | #64748B | 投稿画面のステップタイトル用 |
